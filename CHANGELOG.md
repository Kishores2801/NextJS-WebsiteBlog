# Changelog

## [Unreleased]

### Added
- **Design System**: Implemented a new "Clean & Modern" design system inspired by Gestalt principles.
    - Refined color palette with better contrast and "appealing" tones.
    - Standardized spacing and radius tokens.
    - Mobile-first responsive typography.
- **SEO**: Added JSON-LD structured data, dynamic Sitemap, and RSS feed.
- **Performance**: Moved data fetching to Server Components for better LCP.
- **Features**: Added Sanity Preview Mode support (Drafts vs Published).

### Changed
- **Global CSS**: Refactored `globals.css` to use organized CSS variables and removed legacy styles.
- **Tailwind Config**: Updated `tailwind.config.js` with new theme tokens and animations.
- **Skills Section**: Redesigned to use a modern "Floating Pill" / Bento Grid layout.
- **Project Page**: Converted to Server Component for better SEO.
